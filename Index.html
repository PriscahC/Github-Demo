Github Basics

For beginners, the easiest way to use Github is by downloading Github Desktop

The following are the steps for Setting up Github:

    • Download, install, and run Github on your desktop
      Create an account on Github
      Once in the Github desktop, Go to files, Option, and sign in to your github account
      Confirm that Git has your email and Username
      Navigate to repositories and create a new repository
    • >

    Great! You are now ready to Use Github!
